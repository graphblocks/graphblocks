# graphblocks

Don't reinvent wheel.

This is the provider-neutral GraphBlocks metapackage. It installs the stable
foundation package set used by the default local runtime and conformance
profiles.

This package intentionally owns no Python import package. `graphblocks-core`
provides the `graphblocks` import namespace.

Source and documentation:

- https://github.com/graphblocks/graphblocks
- https://github.com/graphblocks/graphblocks/tree/main/docs
